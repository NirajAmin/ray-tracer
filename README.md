# RayTracer

## Installation

Create the empty output and build folders

Use the provided makefile to run the project  
`make`  
`make run`

## RayTracer features

- A camera with configurable position, orientation, and field of view  
- Anti-aliasing  
- Ray/sphere intersections 
- Ray/triangle intersections  
- The ability to load textures
- Textured spheres and triangles  
- The ability to load and render triangle meshes (.obj)  
- A spatial subdivision acceleration structure (Bounding volume hierarchy)
- Specular, diffuse, and dielectric materials
- Emissive materials
- Volume rendering
- Quads
- Motion blur
- Defocus blur/depth of field
- Object instancing
- Perlin noise
- Importance sampling
- Parallelization
